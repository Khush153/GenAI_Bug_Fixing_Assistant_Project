## üé® User Guide

### How to Use the Tool

1. **Paste Your Code:**
   - Paste the buggy code you want to analyze into the provided text box.

2. **Get Explanations:**
   - Click the ‚ÄúAnalyze‚Äù button to receive a natural language explanation of the bug.

3. **View Fix Suggestions:**
   - The tool will display suggested fixes for your code.
   
4. **Copy and Fix:**
   - You can copy the suggested code and make the changes in your own development environment.

### Example Usage

- **Input Code:**
  ```python
  def add_numbers(a, b):
      return a + b
  print(add_numbers(5, "10"))
  ```

- **Generated Explanation:**
  "The function is trying to add an integer and a string, which will cause a type error."

- **Suggested Fix:**
  ```python
  def add_numbers(a, b):
      return a + int(b)
  ```

## ü§ñ Technical Details

### How It Works

The GenAI Bug Fixing Assistant uses Python and Flask to run a web-based interface. It simulates how large language models analyze code and generate helpful responses. The core logic is based on rule-based reasoning, which enables it to offer insightful bug explanations and fixes.
